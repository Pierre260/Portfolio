SAE PORTFOLIO — Portfolio

🎯 Objectifs

Cette compétence avait pour objectif de concevoir et de présenter une architecture sécurisée de système d'information dans le cadre du SAE Portfolio. Il s'agissait de mettre en valeur l'ensemble des projets et travaux pratiques réalisés durant les 3 années de BUT Réseaux & Télécommunications, parcours Cybersécurité, en proposant un support web structuré, accessible et représentatif des compétences acquises.

🗂️ Le SAE Portfolio

Le Portfolio est une SAE transversale qui consiste à concevoir et déployer un site web personnel regroupant l'ensemble des ressources, projets et expériences réalisés au cours de la formation. Ce projet m'a permis de :

  • Structurer et organiser mes compétences de manière claire et professionnelle
  • Concevoir une interface web complète en HTML et CSS
  • Mettre en ligne le site via GitHub Pages
  • Présenter des architectures et solutions techniques de manière pédagogique
  • Valoriser mon parcours en cybersécurité auprès de futurs employeurs

🛠️ Technologies utilisées

  • HTML5 / CSS3
  • JavaScript (animations canvas)
  • GitHub & GitHub Pages (hébergement)
  • Google Docs / Google Sheets (intégration de rapports)
  • Git (gestion de versions)
Portfolio • GitHub Pages • HTML/CSS

🔍 Compétences développées

  • Concevoir une architecture web structurée et sécurisée
  • Déployer un site statique sur GitHub Pages
  • Organiser et hiérarchiser des contenus techniques
  • Présenter des projets de manière claire et professionnelle
  • Appliquer des bonnes pratiques de développement front-end
  • Gérer un projet en autonomie sur la durée

📝 Mon retour d'expérience

Les principales difficultés ont été l'hébergement sur GitHub Pages — gérer les chemins relatifs et l'arborescence des fichiers — ainsi que les fautes de frappe et erreurs d'indentation dans le code, parfois longues à localiser.

Ce projet m'a appris à être plus rigoureux dans l'écriture du code et dans la gestion d'un déploiement web, des compétences utiles dans le domaine des réseaux et de la cybersécurité.
