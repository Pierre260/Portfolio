SAE 3.02 - Développer des applications communicantes

📌 Objectif du projet

Ce projet consiste à développer un serveur offrant un service de recherche de mots dans différents types de documents. Les clients peuvent interroger le serveur afin de vérifier la présence d’un mot donné dans des fichiers texte (TXT), HTML, PDF et des tableurs Excel (XLSX).

Le serveur fournit également des informations supplémentaires utiles, telles que les lignes où le mot a été trouvé ainsi que le contexte textuel entourant chaque occurrence.

⚙️ Fonctionnalités

  • Architecture client-serveur basée sur les sockets
  • Recherche de mots dans les fichiers TXT, HTML, PDF et XLSX
  • Affichage des numéros de ligne des occurrences
  • Extraction du contexte autour du mot recherché
  • Code Python clair, modulaire et conforme à la norme PEP 8

🛠 Technologies utilisées

  • Python 3
  • Programmation réseau (sockets)
  • Bibliothèque PyPDF2
  • Bibliothèque OpenPyXL
  • HTML / CSS

📁 Organisation du projet

  • server/ : logique du serveur et modules de recherche
  • client/ : application cliente
  • documents/ : fichiers utilisés pour les recherches
  • web/ : interface de présentation du projet

🚀 Utilisation

  1. Lancer le serveur
  2. Démarrer le client
  3. Saisir le mot à rechercher
  4. Sélectionner le type de document
  5. Consulter les résultats retournés par le serveur

J'ai particulièrement appris à :

Cette expérience m'a appris à être plus rigoureux dans l'écriture du code et dans la planification de mon projet. Elle m’a aussi permis de mieux comprendre le travail .
