SAE3.03 : Infrastructure Multi-sites

🎯 Objectif du projet

Ce projet consiste à concevoir et déployer une infrastructure réseau complète pour une entreprise disposant d’un siège et d’une succursale. L’objectif est de garantir une communication sécurisée, performante et évolutive entre plusieurs sites.

🛠 Activités réalisées

  • Conception de l’architecture réseau multi-sites
  • Mise en place d’un Active Directory
  • Configuration des services DNS, mail, proxy
  • Déploiement d’un serveur Web et base de données
  • Mise en place d’un serveur Syslog
  • Segmentation réseau (VLAN)
  • Sécurisation des accès (SSH, filtrage)
  • Réplication des données entre sites
  • Tests et validation des flux réseau

🌐 Architecture

  • ADMIN : gestion des équipements
  • PERSONNEL : postes utilisateurs
  • PRODUCTION : isolé
  • VIDEO : trafic prioritaire
  • WIFI : sécurisé + invité

La succursale est reliée au siège, et tout le trafic Internet passe par le proxy centralisé.
